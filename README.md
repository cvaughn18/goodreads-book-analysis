# goodreads-book-analysis
Analysis of 10,000+ books from Kaggle's Goodreads dataset using Power BI.
## Key Findings
- Twilight dominates popularity with 4.6 million ratings
- Dataset shows surge in book publications, peaking in 2006.
- Most books cluster around 3.8 - 4.2 star ratings
- Different books emerge when looking at text reviews vs. total ratings

## Tools Used
- Power BI Desktop
- Dataset: Kaggle Goodreads Books

## Visualizations
- Most popular books by ratings
- Publication trends over time (1980-present)
- Rating distribution analysis
- Most reviewed books by text reviews
